Spectroscopic selection rules for a diatomic molecule

Pure rotational transitions

ΔJ=+1,+↔−;ΔΛ=0

Requires a permanent electric dipole moment

Rotational-vibrational spectroscopy

ΔJ=±1,+↔−;ΔΛ=0

Δυ=1,(2,3..weaker)

Requires(∂μz/∂q)e≠0

Raman spectroscopy

ΔJ=0,±2,+↔+,−↔−;ΔΛ=0

Δυ=1,(2,3..weaker)

Requires(∂α/∂q)e≠0

Electronic spectroscopy

ΔJ=0,±1,+↔−;ΔΛ=0 or±1, g↔u

∑+−∑+,∑−−∑−;Δs=0a

Δv=any​ value (Frank-Condon principle)

Photoelectron spectroscopy

ΔJ=0,±1,±2±3,…;ΔΛ no restriction

Δ(2s+1)=±1

Δv=any​ value (Frank-Condon principle)

Allowed photoionization processes are one-electron transitions

a Not a strict selection rule
